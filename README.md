# Онлайновий генератор беджів
Просто введіть своє ім'я, прізвище, оберіть посаду, введіть номер телефону (у форматі 0ХХХХХХХХХ), отримайте знімок з веб-камери або перетягніть своє фото й надрукуйте свій бедж!
